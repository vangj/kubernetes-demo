# minikube

To [install](https://kubernetes.io/docs/tasks/tools/install-minikube/) minikube, type in the following.

```bash
brew cask install minikube
```

To start minikube.

```bash
minikube start
```

To stop minikube.

```bash
minikube stop
```

To delete minikube.

```bash
minikube delete
```

To get to the dashboard.

```bash
minikube dashboard
```
